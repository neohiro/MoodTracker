<img width="1097" height="879" alt="image" src="https://github.com/user-attachments/assets/970b4df8-0cf4-45bc-be8c-0422ed02a7e7" />

## Features
- **Daily Mood Logging:** Record your emotional state with an easy-to-use interface.
- **Data Visualization:** View your mood history through beautifully generated charts (powered by Matplotlib).
- **Secure:** Optional password protection to keep your diary entries private.
- **Customizable:** Toggle between Light and Dark mode for comfortable journaling.
- **Export Options:** Export your data seamlessly for backups or personal analysis.

## Getting Started
You can use the standalone executable provided in the Releases tab for your operating system.
Alternatively, you can run from source by installing `matplotlib` and `numpy`:
```bash
pip install matplotlib numpy
python MoodDiary.PY
```
